
### For Loop
Syntax:
```ho
for variable_name from start to end:
    code_block
```

Example:
```ho
for i from 1 to 5:
    print(i)
```

#### While Loop
Syntax:
```ho
variable = initial_value
while condition:
    code_block
    variable = variable + 1  # Increment or decrement the variable
```

Example:
```ho
num = 0
while num < 5:
    print(num)
    num = num + 1
```
